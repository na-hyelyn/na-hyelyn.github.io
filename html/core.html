비즈파인더 (사업자 조회 및 마케팅 솔루션 웹사이트)

비즈파인더 바로가기

비즈파인더는 사업자 등록 및 조회 기능을 기반으로 고객 분석 및 마케팅 타겟팅을 제공하는 서비스형 웹사이트로, 해당 프로젝트에서 퍼블리싱 전반을 담당하였습니다. HTML, CSS, JavaScript, jQuery를 활용하여 전체 화면 구조를 설계하고, 사업자 조회 페이지에서 조회 항목별로 결과가 동적으로 노출되는 프론트엔드 로직을 직접 구현하였습니다. 조회 기능의 화면 흐름과 사용자 인터랙션을 스크립트로 구성한 후, PHP 백엔드 개발자에게 연동을 고려한 형태로 전달하여 협업을 원활하게 진행하였습니다. 이 과정에서 데이터 전달 방식과 화면 처리 로직을 사전에 조율함으로써 개발 효율성을 높였습니다. 또한 회원가입 페이지의 입력값 유효성 검사 로직을 구현하여 사용자 입력 오류를 사전에 방지하였으며, 전반적인 페이지를 반응형 웹으로 구현하여 다양한 디바이스 환경에서도 기능과 레이아웃이 안정적으로 동작하도록 구성하였습니다.

딥시크 (서비스형 웹사이트 프로토타입)

딥시크 바로가기

딥시크는 실제 서비스 창출을 목적으로 기획·운영된 웹사이트로, 실제 딥시크 서비스와 동일한 사용자 흐름과 UI를 구현한 프로젝트입니다. 해당 프로젝트에서 프론트엔드 전반의 스크립트 처리를 단독으로 담당하였으며, HTML, CSS, JavaScript, jQuery를 활용해 서비스 화면을 정교하게 구현하였습니다. 특히 채팅 서비스와 관련하여 채팅 메시지가 출력되는 형태, 채팅 도중 중단되는 효과, 화면 전환 흐름 등을 PHP와 스크립트를 활용해 세밀하게 조정하였으며, 사용자 행동에 따라 자연스럽게 반응하는 인터랙션을 구현하였습니다. 또한 전체 페이지를 반응형 웹으로 구현하여 디바이스 환경에 따른 UI 깨짐 없이 안정적인 사용 경험을 제공하였으며, 실제 서비스 운영을 고려한 구조 설계를 통해 유지보수와 확장성을 함께 고려한 퍼블리싱 작업을 수행하였습니다.

협업 및 업무 방식

모든 프로젝트에서 PHP 백엔드 개발자와의 긴밀한 협업을 통해 업무를 진행하였으며, 단순 퍼블리싱에 그치지 않고 PHP 로직을 이해하고 응용하며 실제 코드 작성에도 적극 참여하였습니다. 유지보수와 확장성을 고려한 코드 작성, 표준을 준수한 마크업, 명확한 커뮤니케이션을 바탕으로 안정적인 프로젝트 진행과 완성도 높은 결과물을 도출하였습니다.
